Expand description

OpenMP directive type

This enum covers all standard OpenMP directives from the 5.2 specification. Each directive is represented as a unique variant.

§Examples

let kind = DirectiveKind::Parallel;
assert_eq!(kind.to_string(), "parallel");

let kind = DirectiveKind::ParallelFor;
assert_eq!(kind.to_string(), "parallel for");

§Learning: Large Enum with Clear Organization

This enum demonstrates organizing a large number of variants (70+) into logical categories using comments and grouping.

pub const fn name_for_language(self, language: Language) -> &'static str

Get the language-specific directive name

Returns the appropriate directive name for the given language context. For C/C++, returns forms like “for”, “parallel for”. For Fortran, returns forms like “do”, “parallel do”.

This method is used for cross-language translation of directives.

§Examples
assert_eq!(DirectiveKind::For.name_for_language(Language::C), "for");
assert_eq!(DirectiveKind::For.name_for_language(Language::Fortran), "do");
assert_eq!(DirectiveKind::ParallelFor.name_for_language(Language::C), "parallel for");
assert_eq!(DirectiveKind::ParallelFor.name_for_language(Language::Fortran), "parallel do");

pub const fn as_str(self) -> &'static str

Get the canonical directive name as a string

Returns the standard C/C++ form of the directive name. This is the same as the Display implementation.
