Language Support

C/C++ Syntax

#pragma omp parallel for num_threads(4) schedule(static)
for (int i = 0; i < n; i++) {
    process(i);
}

‚úÖ Supported: All C/C++ #pragma omp directives

Fortran Syntax

!$omp parallel do schedule(dynamic)
do i = 1, n
    call process(i)
end do
!$omp end parallel do

‚úÖ Supported: Free-form (!$omp), Fixed-form (c$omp, *$omp)
‚úÖ Case insensitive: !$OMP, !$Omp, !$omp all work

Unsupported Features

ROUP focuses on directive parsing, not runtime semantics. The following are out of scope:

‚ùå Runtime execution - ROUP parses directives, doesn't execute them
‚ùå Code transformation - No AST rewriting or code generation
‚ùå Semantic validation - Doesn't check if directives make logical sense
‚ùå Context analysis - Doesn't validate directive placement in code

Use ROUP for:

  • ‚úÖ Static analysis tools
  • ‚úÖ Code documentation generators
  • ‚úÖ IDE syntax highlighting
  • ‚úÖ Linting and code quality checks
  • ‚úÖ Migration tools
  • ‚úÖ Research prototypes

Don't use ROUP for:

  • ‚ùå Compiling OpenMP to executables (use GCC/Clang)
  • ‚ùå Runtime task scheduling (use OpenMP runtime)
  • ‚ùå Performance profiling (use performance tools)
